# Crush-CN 安装部署

## 安装方式概览

Crush-CN 支持多种安装方式，包括包管理器安装、二进制安装、源码编译安装和 Docker 部署。

| 安装方式 | 适用场景 | 难度 |
|----------|----------|------|
| 包管理器 | 日常使用 | 简单 |
| 二进制安装 | 无包管理器的系统 | 简单 |
| 源码编译 | 开发者、自定义构建 | 中等 |
| Docker | 容器化环境 | 中等 |

## 包管理器安装

### macOS / Linux - Homebrew

```bash
# 添加 Charm tap
brew tap charmbracelet/tap

# 安装 Crush
brew install charmbracelet/tap/crush

# 更新
brew upgrade crush
```

### Linux - AUR (Arch Linux)

```bash
# 使用 yay 安装二进制包
yay -S crush-bin

# 或从源码构建
yay -S crush-git
```

### Linux - Nix

```bash
# 直接运行
nix run github:numtide/nix-ai-tools#crush

# 或通过 NUR 安装
nix-channel --add https://github.com/nix-community/NUR/archive/main.tar.gz nur
nix-channel --update
nix-shell -p '(import <nur> { pkgs = import <nixpkgs> {}; }).repos.charmbracelet.crush'
```

### FreeBSD

```bash
# 使用 pkg 安装
pkg install crush
```

### Windows - Winget

```powershell
# 使用 Winget 安装
winget install charmbracelet.crush
```

### Windows - Scoop

```powershell
# 添加 Charm bucket
scoop bucket add charm https://github.com/charmbracelet/scoop-bucket.git

# 安装 Crush
scoop install crush

# 更新
scoop update crush
```

### NPM

```bash
# 全局安装
npm install -g @charmland/crush

# 更新
npm update -g @charmland/crush
```

## 二进制安装

### 下载预编译二进制

从 GitHub Releases 下载适合您系统的二进制文件：

```bash
# 查看 Releases 页面
# https://github.com/purpose168/crush-cn/releases

# Linux amd64 示例
wget https://github.com/purpose168/crush-cn/releases/download/v0.1.0/crush_0.1.0_Linux_x86_64.tar.gz
tar -xzf crush_0.1.0_Linux_x86_64.tar.gz
sudo mv crush /usr/local/bin/

# macOS arm64 示例
wget https://github.com/purpose168/crush-cn/releases/download/v0.1.0/crush_0.1.0_Darwin_arm64.tar.gz
tar -xzf crush_0.1.0_Darwin_arm64.tar.gz
sudo mv crush /usr/local/bin/

# Windows amd64 示例 (PowerShell)
Invoke-WebRequest -Uri "https://github.com/purpose168/crush-cn/releases/download/v0.1.0/crush_0.1.0_Windows_x86_64.zip" -OutFile "crush.zip"
Expand-Archive -Path "crush.zip" -DestinationPath "."
Move-Item -Path "crush.exe" -Destination "C:\Windows\"
```

### 验证安装

```bash
# 检查版本
crush --version

# 查看帮助
crush --help
```

## 源码编译安装

### 前置条件

- Go 1.25.5+
- Git
- Task（可选）

### 编译步骤

```bash
# 克隆仓库
git clone https://github.com/purpose168/crush-cn.git
cd crush-cn

# 获取标签
git fetch --tags

# 安装依赖
go mod download

# 方法一：使用 Taskfile
task install

# 方法二：直接使用 Go
go install -ldflags="-X github.com/purpose168/crush-cn/internal/version.Version=$(git describe --long)" .

# 验证安装
crush --version
```

### 自定义构建

```bash
# 禁用 CGO 构建（推荐）
CGO_ENABLED=0 go build -ldflags="-s -w" -o crush .

# 启用实验性功能
GOEXPERIMENT=greenteagc go build -o crush .

# 交叉编译
GOOS=linux GOARCH=arm64 CGO_ENABLED=0 go build -o crush-linux-arm64 .
```

## Docker 部署

### 使用预构建镜像

```bash
# 拉取镜像
docker pull ghcr.io/purpose168/crush-cn:latest

# 运行容器
docker run -it --rm \
    -v ~/.config/crush:/root/.config/crush \
    -v $(pwd):/workspace \
    ghcr.io/purpose168/crush-cn:latest
```

### 自定义 Dockerfile

```dockerfile
# 构建阶段
FROM golang:1.25.5-alpine AS builder

WORKDIR /app
COPY . .

RUN go mod download
RUN CGO_ENABLED=0 go build -ldflags="-s -w" -o crush .

# 运行阶段
FROM alpine:latest

RUN apk add --no-cache git

WORKDIR /workspace
COPY --from=builder /app/crush /usr/local/bin/crush

ENTRYPOINT ["crush"]
```

### 构建和运行

```bash
# 构建镜像
docker build -t crush-cn:latest .

# 运行容器
docker run -it --rm \
    -v ~/.config/crush:/root/.config/crush \
    -v $(pwd):/workspace \
    crush-cn:latest
```

### Docker Compose

```yaml
version: '3.8'

services:
  crush:
    image: ghcr.io/purpose168/crush-cn:latest
    container_name: crush
    volumes:
      - ~/.config/crush:/root/.config/crush
      - .:/workspace
    working_dir: /workspace
    stdin_open: true
    tty: true
```

## Shell 补全

### Bash

```bash
# 生成补全脚本
crush completion bash > ~/.local/share/bash-completion/completions/crush

# 或添加到 .bashrc
echo 'source <(crush completion bash)' >> ~/.bashrc
```

### Zsh

```bash
# 生成补全脚本
crush completion zsh > "${fpath[1]}/_crush"

# 或添加到 .zshrc
echo 'source <(crush completion zsh)' >> ~/.zshrc
```

### Fish

```bash
# 生成补全脚本
crush completion fish > ~/.config/fish/completions/crush.fish
```

### PowerShell

```powershell
# 生成补全脚本
crush completion powershell | Out-File -Encoding UTF8 $PROFILE

# 或手动添加
crush completion powershell >> $PROFILE
```

```bash
# Homebrew
brew upgrade crush

scoop update crush

npm update -g @charmland/crush

yay -S crush-bin

task install
```

## 卸载

### 包管理器卸载

```bash
# Homebrew
brew uninstall crush

# Scoop
scoop uninstall crush

# NPM
npm uninstall -g @charmland/crush

# FreeBSD
pkg delete crush
```

### 手动卸载

```bash
# 删除二进制文件
rm /usr/local/bin/crush

# 删除配置文件（可选）
rm -rf ~/.config/crush

# 删除数据文件（可选）
rm -rf ~/.local/share/crush

# 删除补全脚本
rm ~/.local/share/bash-completion/completions/crush
rm ~/.config/fish/completions/crush.fish
rm "${fpath[1]}/_crush"
```

## 平台特定说明

### Linux

- 大多数 Linux 发行版可以直接使用预编译二进制
- 对于 ARM 设备（如树莓派），使用 arm64 或 arm 版本
- 某些发行版可能需要安装额外的依赖

### macOS

- 支持 Intel (amd64) 和 Apple Silicon (arm64)
- 首次运行可能需要在系统偏好设置中允许

### Windows

- 支持 PowerShell 和 WSL
- 建议使用 Windows Terminal 获得最佳体验
- 某些功能在 CMD 中可能受限

### FreeBSD / OpenBSD / NetBSD

- 使用预编译二进制或从源码编译
- 可能需要安装额外的依赖

### Android (Termux)

```bash
# 在 Termux 中安装
pkg install crush

# 或使用 dpkg
dpkg -i crush_*.deb
```
