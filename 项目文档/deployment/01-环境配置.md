# Crush-CN 环境配置

## 开发环境要求

### Go 语言环境

Crush-CN 使用 Go 1.25.5 开发，需要安装 Go 语言环境。

#### Linux/macOS 安装 Go

```bash
# 下载 Go 1.25.5
wget https://go.dev/dl/go1.25.5.linux-amd64.tar.gz

# 解压到 /usr/local
sudo tar -C /usr/local -xzf go1.25.5.linux-amd64.tar.gz

# 设置环境变量
export PATH=$PATH:/usr/local/go/bin
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin

# 验证安装
go version
```

#### macOS 使用 Homebrew 安装

```bash
brew install go
```

#### Windows 安装 Go

1. 下载 Go 安装程序：https://go.dev/dl/go1.25.5.windows-amd64.msi
2. 运行安装程序
3. 重启命令提示符或 PowerShell
4. 验证安装：
   ```powershell
   go version
   ```

## 环境变量

### Go 环境变量

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `GOPATH` | Go 工作目录 | `$HOME/go` |
| `GOROOT` | Go 安装目录 | `/usr/local/go` |
| `GOPROXY` | Go 模块代理 | `https://proxy.golang.org,direct` |
| `GOSUMDB` | Go 校验和数据库 | `sum.golang.org` |
| `CGO_ENABLED` | 是否启用 CGO | `1` |
| `GOEXPERIMENT` | Go 实验性功能 | `greenteagc` |

### 项目环境变量

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `CRUSH_PROFILE` | 启用性能分析 | `false` |
| `CRUSH_CONFIG_PATH` | 配置文件路径 | `~/.config/crush` |

### 配置示例

**Linux/macOS**：
```bash
# 添加到 ~/.bashrc 或 ~/.zshrc
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin
export PATH=$PATH:/usr/local/go/bin

# 设置 Go 模块代理（中国用户推荐）
export GOPROXY=https://goproxy.cn,direct

# 启用 Go 实验性功能
export GOEXPERIMENT=greenteagc
```

**Windows PowerShell**：
```powershell
# 添加到 $PROFILE
$env:GOPATH = "$env:USERPROFILE\go"
$env:PATH += ";$env:GOPATH\bin"
$env:GOPROXY = "https://goproxy.cn,direct"
$env:GOEXPERIMENT = "greenteagc"
```

## IDE 配置

### VS Code

1. 安装 Go 扩展
2. 安装 Go Nightly 扩展（可选）
3. 配置 settings.json：

```json
{
    "go.useLanguageServer": true,
    "go.lintTool": "golangci-lint",
    "go.lintOnSave": "package",
    "go.formatTool": "gofumpt",
    "go.testFlags": ["-v", "-race"],
    "go.buildFlags": ["-tags=integration"],
    "[go]": {
        "editor.formatOnSave": true,
        "editor.codeActionsOnSave": {
            "source.organizeImports": true
        }
    }
}
```

### GoLand

1. 打开 Settings/Preferences
2. Go > GOROOT：设置 Go 安装路径
3. Go > Go Modules：启用 Go modules
4. Tools > File Watchers：添加 gofumpt
5. Editor > Inspections：启用 golangci-lint

## 常见问题

### 1. Go 版本不匹配

**问题**：
```
go: go.mod requires go >= 1.25.5
```

**解决方案**：
```bash
# 更新 Go 到最新版本
go install golang.org/dl/go1.25.5@latest
go1.25.5 download

# 设置为新版本
go env -w GOTOOLCHAIN=go1.25.5
```

### 2. 依赖下载失败

**问题**：
```
go: github.com/purpose168/crush-cn: reading at revision: unknown revision
```

**解决方案**：
```bash
# 设置代理
export GOPROXY=https://goproxy.cn,direct

# 清理缓存
go clean -modcache

# 重新下载
go mod download
```

### 3. CGO 相关问题

**问题**：
```
# runtime/cgo
/usr/bin/ld: cannot find -lc
```

**解决方案**：
```bash
# 禁用 CGO
export CGO_ENABLED=0

# 或者安装 C 编译器
# Ubuntu/Debian
sudo apt-get install build-essential

# macOS
xcode-select --install

# Windows - 安装 MinGW 或 MSVC
```

### 4. Task 命令找不到

**问题**：
```
task: command not found
```

**解决方案**：
```bash
# 确保 GOPATH/bin 在 PATH 中
export PATH=$PATH:$(go env GOPATH)/bin

# 或者使用绝对路径
~/go/bin/task
```

## 验证环境

运行以下命令验证开发环境是否正确配置：

```bash
# 检查 Go 版本
go version
# 输出: go version go1.25.5 linux/amd64

# 检查 Go 环境
go env GOPATH GOROOT GOPROXY

# 检查依赖
go mod verify

# 检查 Task
task --version

# 检查 golangci-lint
golangci-lint version

# 尝试构建
task build
```
