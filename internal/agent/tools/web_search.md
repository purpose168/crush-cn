使用 DuckDuckGo 搜索网络并返回搜索结果。

<usage>
- 提供搜索查询以在网络上查找信息
- 返回包含标题、URL 和摘要的搜索结果列表
- 使用此工具查找相关网页，然后使用 web_fetch 获取完整内容
</usage>

<parameters>
- query: 搜索查询字符串（必填）
- max_results: 返回的最大结果数（默认：10，最大：20）
</parameters>

<tips>
- 使用具体、有针对性的搜索查询以获得更好的结果
- 获取结果后，使用 web_fetch 获取相关页面的完整内容
- 组合多个搜索以收集全面的信息
</tips>
