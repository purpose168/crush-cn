# 项目说明文档

## 1. 项目来源声明

本项目是基于 Charmbracelet, Inc. 开发的 crush 项目进行二次开发的衍生项目。我们严格遵循原始项目的开源协议，并在其基础上进行了功能增强和本地化改进。

## 2. 原始项目信息

- **项目名称**: crush
- **原始项目链接**: https://github.com/charmbracelet/crush
- **原始许可证**: Functional Source License, Version 1.1, MIT Future License (FSL-1.1-MIT)
- **版权所有者**: Charmbracelet, Inc. (2025-2026)

## 3. 二次开发说明

### 3.1 开发目的
本衍生项目旨在：
- 对原始项目进行中文本地化，使其更适合中文用户使用
- 针对中国用户的使用习惯进行界面优化
- 保持与原始项目的功能兼容性
- 遵循开源协议的所有要求

### 3.2 主要修改内容
- 将代码中的英文注释翻译为中文
- 优化中文界面显示效果
- 保持核心功能与原始项目一致
- 确保所有修改符合原始许可证的要求
