## 项目核心信息摘要

### 项目定位
Crush 是一个强大的终端 AI 辅助编程工具,提供智能代码助手功能。

### 技术栈
- **编程语言**: Go 1.25.5
- **UI框架**: BubbleTea v2 + Bubbles v2 + Lipgloss v2
- **AI框架**: Fantasy + Catwalk
- **数据库**: SQLite + sqlc
- **构建工具**: Taskfile + GoReleaser

### 核心功能
- 多AI模型支持(OpenAI, Anthropic, Google等)
- LSP集成(代码诊断、引用查询)
- MCP支持(工具扩展)
- 会话管理和历史记录
- 权限控制和安全执行

### 架构特点
- 分层架构设计
- 事件驱动通信
- 模块化设计
- 接口隔离原则
